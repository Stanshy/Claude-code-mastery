## Course Structure: Five-Layer Evolution Model

This course is built on David Chu's Five-Layer Evolution Model. Each module corresponds to an evolution layer:

```
Layer 1  Conversational  →  Module 1: Introduction to Claude Code
Layer 2  Configured      →  Module 2: Configuration
Layer 3  Automated       →  Module 3: Hooks / Skills / MCP
Layer 4  Orchestrated    →  Module 4: Subagents / Context Management / Parallel Dev
Layer 5  Autonomous      →  Module 5: Agent Teams / Headless / SDK
Methodology             →  Module 6: Harness Engineering
Case Studies            →  Module 7: Boris Cherny / gstack / Open-Source Cases
Reference               →  Module 8: Cheatsheet + Resource Index
```
