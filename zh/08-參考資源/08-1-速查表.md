## 表格 4：Hook 事件速查

| 事件 | 觸發時機 | 可阻擋 | 常見用途 |
|------|---------|--------|---------|
| `SessionStart` | 會話開始/恢復 | 否 | 載入上下文 |

| `PreCompact` | 壓縮前 | 否 | 備份 |

## 表格 5：權限語法速查

| 模式 | 範例 | 說明 |
|------|------|------|
| 精確匹配 | `Bash(npm run test)` | 只匹配此命令 |
| 前綴通配 | `Bash(npm *)` | npm 開頭的所有命令 |
| 後綴通配 | `Bash(* test)` | test 結尾的所有命令 |
| 路徑模式 | `Edit(.env*)` | .env 開頭的檔案 |
| MCP 工具 | `mcp__github__*` | GitHub MCP 所有工具 |
| Subagent | `Agent(Explore)` | 特定 Subagent |

## 表格 6：常見問題排除

| 問題 | 原因 | 解法 |
|------|------|------|
| `command not found: claude` | 未安裝或不在 PATH | 重新安裝，檢查 PATH |
| `You need to authenticate` | 未登入 | `claude auth login` |
| Claude 忽略 CLAUDE.md 規則 | 檔案太長或規則太模糊 | 精簡到 200 行，用強調語 |
| Hook 未觸發 | matcher 拼寫錯誤 | `/hooks` 確認，注意大小寫 |
| Hook 無限迴圈 | Stop Hook 未檢查 `stop_hook_active` | 加 `stop_hook_active` 檢查 |
| MCP 連線失敗 | Token 無效或格式錯誤 | `/mcp` 檢查狀態 |
| Context 過載 | 會話太長 | `/clear` 或 `/compact` |
| JSON parse error | settings.json 語法錯誤 | `node -e "JSON.parse(...)"` 驗證 |
